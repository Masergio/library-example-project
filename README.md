# Library web application

Example project for PY-105 students.


## Project Setup

### Install dependencies
```shell
pip install -r requirements.txt
```

### Apply migrations
```shell
python manage.py migrate
```

### Create admin user
```shell
python manage.py createsuperuser
```

### Run local server
```shell
python manage.py runserver [8000]
```
